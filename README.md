# Population VCS Lab — вариант 5

## Описание проекта

Данный проект выполнен в рамках лабораторной работы по изучению системы контроля версий Git и организации командной разработки.  
Реализован программный продукт с графическим пользовательским интерфейсом для анализа численности населения Российской Федерации и статистического прогнозирования методом скользящей средней.

Проект реализует **вариант 5** задания:
> Анализ данных о численности населения России за последние 15 лет с построением графиков, вычислением годовых изменений и прогнозированием на N лет методом скользящей средней.

---

## Функциональные возможности

Программа предоставляет следующие функции:

- загрузка файла с данными о численности населения (CSV / XLSX / JSON);
- отображение данных в табличном виде;
- вычисление абсолютного и процентного изменения численности населения по годам;
- определение года максимального процента прироста и убыли населения;
- построение графика фактической численности населения;
- построение прогноза численности населения на N лет методом скользящей средней;
- визуальное отображение прогноза на графике.

---

## Метод прогнозирования

Для прогнозирования используется метод **экстраполяции по скользящей средней**.

Скользящая средняя рассчитывается по формуле:

\[
\overline{x} = \frac{1}{n} \sum_{i=1}^{n} x_i
\]

где:
- `n` — размер окна скользящей средней (задаётся пользователем),
- `x_i` — значения численности населения за предыдущие периоды.

Прогнозируемое значение принимается равным средней величине за последний интервал времени.

## Пользовательский интерфейс

Графический пользовательский интерфейс реализован с использованием библиотеки **Tkinter** и включает:

- кнопку выбора файла данных;
- поля ввода параметров `n` (размер окна) и `N` (число лет прогноза);
- кнопку запуска расчёта;
- область вывода итогов анализа;
- отдельные окна для таблицы данных и графика.

## Запуск программы

1. Убедитесь, что установлен Python версии 3.10 или выше.
2. Установите необходимые библиотеки:
   pip install pandas matplotlib
Перейдите в корневую папку проекта.

Запустите программу командой:
python -m app.main

Организация командной работы (Git)

Работа выполнена с имитацией командной разработки:

использованы ветки:

main — основная ветка проекта;

dev — интеграционная ветка;

отдельные feature-ветки для каждого участника;

каждый участник выполнил не менее 5 осмысленных коммитов;

изменения вносились через Pull Request в ветку dev;

для Pull Request выполнялось review кода с комментариями;

после завершения разработки выполнено слияние ветки dev в main.

История коммитов и ветвлений доступна в онлайн-репозитории и подтверждает командный характер работы.
